
Step 1: create an Backend folder
Step 2: npm init -y   
Step 3: create an src folder
Step 4: npm i express
Step 5: npm i -D typescript ts-node @types/node @types/express nodemon
Step 6: npx tsc --init

IN tsconfig.json file

Step 7:
 {
  "compilerOptions": {
    "target": "es2020",
    "module": "commonjs",
    "strict": true,
    "esModuleInterop": true,
    "outDir": "./dist",
    "rootDir": "./src",
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules"]
}


Inside src service.ts file

Step 8:
import express, { Application, Request, Response } from 'express';

const app: Application = express();
const PORT = process.env.PORT || 3000;

app.get('/', (req: Request, res: Response) => {
  res.send('Welcome to Express & TypeScript Server!');
});

app.listen(PORT, () => {
  console.log(`Server is running on http://localhost:${PORT}`);
});


Inside package.josn file

Step 9:
"scripts": {
  "start": "node dist/server.js",
  "build": "tsc",
  "dev": "nodemon src/server.ts"
},


Step 10: Run the Application

To run in development mode with auto-restarts: This code is for development in the local network then after the code is completed the following code is written to send the app into production.
npm run dev

To build for production (compiles .ts to .js in the dist folder):
npm run build

To start the compiled production code:
npm start
